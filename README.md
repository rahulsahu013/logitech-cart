# Logitech Cart
A simple shopping cart application built using React and Vite. It allows users to view products, remove/reset products from the cart, and see real-time cart total updates.

# Project Setup
git clone <https://github.com/rahulsahu013/logitech-cart.git>

cd shopping-cart

# Install dependencies
Make sure you have Node.js installed (recommended Node 16+).

npm install

# Start development server
npm run dev

# Local Development Server
After running the app, you will see:

➜  Local:   http://localhost:8080/
